# Sound Effects

Add your sound effect files here:

- `thwip.mp3` - Web shooting sound
- `impact.mp3` - Web impact/attachment sound
- `swing.mp3` - Swinging whoosh sound (loopable)
- `pull.mp3` - Object pulling sound

## Recommended Sound Sources

1. **Freesound.org** - Free sound effects library
2. **Zapsplat.com** - Free sound effects for personal/commercial use
3. **YouTube Audio Library** - Royalty-free sounds
4. **Record your own** - Use a smartphone to record custom sounds

## Sound Specifications

- Format: MP3 or OGG
- Sample Rate: 44.1kHz or 48kHz
- Bit Rate: 128kbps or higher
- Length: 0.5-2 seconds (except swing which should be 2-5 seconds)

The app will work without sounds - they're optional enhancements!
